# vscode-components

Monorepo for VS Code-like React components and a local preview extension.

This repository is currently in alpha. APIs, package contents, and development workflows may change without notice.

## Packages

- `packages/vscode-components`: publishable React component library
- `apps/vscode-components-preview`: VS Code extension for local preview

## Commands

```sh
npm run dev
npm run dev:extension
npm run build
npm run check
```
